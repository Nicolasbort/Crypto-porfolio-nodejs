# Cryptocurrency Portfolio

## Tools

- Node.js
- Vue.js
- Binance API

## Npm Modules

- Express
- Socket.io
- Axios
- Https

## How to use

`npm install`

- Create a .env file in the repository folder and use the .env.example template to fill it

- Open two terminals

#### First Terminal

`cd server && npm run start`

### Second Terminal

`cd client && npm run serve`

- Open http://localhost:8080/
